How to use booking history and cancel booking at frontend

Price package is a function that allows defining payment packages. Customers can purchase packages to pay for services without having to pay through payment gateways such as Paypal, Stripe, ...

To use booking history at frontend, please do follow:

  • Enable Price package in FAT Service Booking -> Setting -> Payment.
  • Create price package
  • Create price package order history. This is page what customer can see our order history
