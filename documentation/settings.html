Settings

FAT Services Booking provide 7 options setting , they are Layout setting, Data source setting, Filter / Sort setting, Pagination setting,Skins & Animation setting, Font & Color setting, and Custom CSS Setting.

Layout Setting

This option use to setting for layout

Data source setting

This option use to build data source for shortcode. In this option, use can choice select portfolio from categories or pick individual portfolio or choice porfolio what create by author

Filter / Sort setting

With setting, use can choice filter by isotope or ajax and choice order display

Pagination setting

Fat Portfolio provide 3 pagination style and pagination color style:

  • Paging number
  • Load more
  • Infinite scroll

Skins & Animation setting

In this option, use can choice item skin (include 10 skin) and item appear animation (include 11 animation)

Font & Color setting

Using this feature, you can add your style sheet for portfolio shortcode:

  • Title font size, font color, font style and font weight
  • Category font size, font color, font style and font weight
  • Category filter font size, font color, font style and font weight
  • Background hover color

Custom CSS setting

In this option, you can add your custom style sheet for shortcode
